Software de inventarios para la panaderia Mary pan en el Tolima

Fecha fin: Diciembre 2022

Fecha inicio: septiembre 2022

Descripción: realizo un software de inventarios para mantener en un servidor privado para uso exclusivo de la panaderia Mary pan del Departamento del Tolima, como desarrollador de software encargado junto con mi compañera de equipo tuvimos que realizar, la maquetación web en Html y css, el diseño del logotipo, el diseñi de imagenes para uso exclusivo de la panaderia, el diseño, creación y administración de la base de datos en sql Mysql, la programación en php, json y javascript para toda la parte funcional y el uso de chart.js para las estadisticas que se muestran del stock en bodega, las ventas y pedidos.
